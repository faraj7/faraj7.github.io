A implementação do conversor AD deste experimento utilizará a estratégia de aproximações sucessivas.

O módulo utiliza um conversor digital-analógico e um comparador para encontrar qual a melhor aproximação para o valor sendo convertido.

Inicialmente ele chuta um possível valor para a resposta da conversão. Este valor então é convertido para sua forma analógica com o conversor DA e é comparado com o sinal de entrada. A partir da resposta do comparador, o módulo de controle busca um novo valor que melhor aproxima o sinal de entrada.

Para encontrar o nível correto utiliza-se o algoritmo da busca binária entre todos os possíveis valores digitais.

A ativação do módulo conversor é realizada em SIM_SCGC6.

A tabela de Multiplexação dos Pinos deve ser consultada de maneira a se conectar PTB1 ao conversor.

Para configurar a operação do conversor, os registradores de controle são: ADC0_CFG1, ADC0_SC2, ADC0_SC3, ADC0_SC1A.

Deve-se consultar o manual de maneira a configurá-los para o funcionamento descrito no Roteiro.

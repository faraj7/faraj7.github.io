Tempo de impressão terminal serial e LCD.

Quanto tempo demora para que o valor do cronômetro seja impresso no terminal serial? E quanto tempo demora para imprimir no LCD? Como estamos trabalhando com um cronômetro de centécimos de segundos, é importante sabermos quanto tempo demora para se imprimir o string do cronômetro no terminal serial a 19200 bps e quanto tempo demora para imprimir o valor do cronômetro no LCD. Desconsiderar os cálculos de conversão de inteiro para ASCII, supor que os strings já estão prontos para imprimir.

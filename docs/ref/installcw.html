INSTALAÇÃO DO IDE, DRIVERS E PLUGIN

O CodeWarrior 10.6 está disponível para download. A versão gratuita (Special Edition) tem total funcionalidade, e a única limitação é o tamanho do código objeto em projetos feitos em C ou C++ (para os processadores Kinetis L é de 64kB). Para projetos totalmente em Assembly, não há limitação. Para baixar o instalador online(700MB), use o seguinte link: CW 10.6

Este instalador exige conexão à internet, para baixar os arquivos de suporte dos processadores.Em um momento da instalação, o usuário deve selecionar as famílias de componentes a serem suportadas. Deve- se selecionar a família Kinetis ARM, para suporte à placa do curso.

Após a instalação do CodeWarrior, deve- se instalar o pacote de drivers (arquivo PEDrivers_install.exe) para acesso ao OpenSDA da placa, antesde se plugar a placa ao computador. Após a instalação, pode- se plugar a placa e aguardar o reconhecimento dos drivers. Finalmente, para se utilizar um console ligado à porta serial dentro da perspectiva de debug do CodeWarrior, é necessário instalar um plugin : RXTXUpdatesiteI200710161945.zip

Para instalar o plugin, deve- se inicialmente descompactar o arquivo. Depois, abra o CodeWarrior e selecionena barra de menu, opção Help - Install New Software, e na janela que se abre clique no botão Add . Clique no botão Local, e navegue até a pasta criada. Depois é só abrir a árvore do TXRX e selecionar APENAS o runtime (desmarque oSDK). Depois é só prosseguir com a instalação.

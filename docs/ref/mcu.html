Neste curso iremos exercitar o desenvolvimento de aplicações no microcontrolador ARM. Este microcontrolador é composto de várias partes e com fabricantes distintos:

  • Shield: placa feita na FEEC - contem o visor LCD, LEDs vermelhos e botões
  • Freedom KL25: placa de desenvolvimento feita pela Freescale
  • KL25P80: microcontrolador 32 bits fabricado pela Freescale com arquitetura ARM
  • Cortex M0+: CPU ARM 32 bits conjunto instruções Thumb2, licenciado pela ARM

A seguir figuras ilustrativas destas partes. Ao longo do curso iremos precisar de informações associadas a cada uma destas partes, assim é importante entendê-las assim como os manuais que contém as informações desejadas.

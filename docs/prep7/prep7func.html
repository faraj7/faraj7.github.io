Como nossos programas irão quase que sempre utilizarem o LCD e UART, seguem as sugestões de funções a serem feitas, de modo que os programas fiquem mais simples de serem acompanhados:
  1. LCD
    1. Inicialização:
      1. void init_GPIO_LCD(); // inicializa as portas do GPIO para controlar LCD
      2. void init_LCD(); // inicializa o LCD, configuração 2 colunas, limpeza do display, etc.
    2. Escrita no LCD
      1. void putchar_LCD(char c); // imprime o código ASCII guardado em c no LCD
        • // Esta função deve setar no início o sinal RS = 1, indicando que se está enviando dados ao LCD
      3. void sendcmd_LCD(char c, int us); // envia código de comando para o LCD e espera us microssegundos
        • // Esta função deve setar no início o sinal RS = 0, indicando que se está enviando comandos para o LCD
    3. Mensagem no LCD
      1. void print_LCD(char *s); // envia mensagem ASCII armazenada no string s para o LCD
  2. Terminal Serial, UART0
    1. Inicialização:
      1. void init_UART(); // configura UART 19200, 8 bits, sem paridade, 1stop 1 start bit
    2. Escrita no terminal serial
      1. void putchar_UART(char c); // imprime o código ASCII guardado em c no terminal serial
    3. Leitura do terminal serial
      1. char getchar_UART(); // leitura de caractere do teclado com retorno imediato<br />// caso o caractere não esteja pronto, retorna 0
    4. Mensagem para o terminal serial
      1. void print_UART( char *s); // imprime string s codificada em ASCII no terminal serial
  3. Botões
    1. Leitura dos botões
      1. int getkey(); // Leitura dos botões, retorna 0 se nada foi apertado, retorna 1, 2 ou 3 dependendo de qual botão foi apertado
